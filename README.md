# Main Pour la Paix - Site Web

Site web de l'association Main Pour la Paix, dédié à la protection de la santé dans les zones de conflit.

## Installation

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Démarrer le serveur de développement :
   ```bash
   npm start
   ```

3. Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Technologies utilisées

- React 18
- Tailwind CSS
- Lucide React (icônes)

## Structure du projet

- `src/MPPWebsite.jsx` : Composant principal du site
- `src/App.js` : Point d'entrée de l'application
- `src/index.js` : Rendu de l'application
- `public/index.html` : Template HTML

## Fonctionnalités

- Page unique avec navigation interne
- Section de dons (placeholder pour intégration Stripe/PayPal)
- Formulaire de bénévolat
- Galerie photos (placeholders)
- Informations sur les partenaires et rapports

Le code est fonctionnel et prêt pour le déploiement.
